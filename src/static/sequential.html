Busca Sequencial

Algoritmo de Busca Linear - O(n)

Como funciona:

A busca sequencial percorre todos os registros da base de dados um por um, comparando cada entrada com o termo de busca. É o método mais simples de implementar, porém apresenta performance inferior em grandes volumes de dados, pois no pior caso precisa verificar todos os 5.000 registros.
